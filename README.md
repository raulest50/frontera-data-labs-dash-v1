# Neo-Miami IoT Dashboard

A futuristic Miami-Vice-inspired smart city environmental dashboard.
Built with:

- React + Vite
- Bun
- Chakra UI v3
- Three.js
- Docker ready

## Install

bun install

## Run Dev Server

bun dev

## Build

bun run build

## Docker Build

docker build -t neo-miami-iot .

## Docker Run

docker run -p 3000:3000 neo-miami-iot
